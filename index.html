2. Information We Collect

2.1 Personal Data

The application does not require users to create an account and does not directly collect personal identifiable information.

However, the following may be processed:

  • Anonymous user ID (generated via Firebase Authentication)
  • Push notification token

2.2 Non-Personal Data

The application may automatically collect:

  • Device type
  • Operating system version
  • Anonymous usage data
  • App performance data

3. Push Notifications

The application may send notifications to inform users about:

  • School announcements
  • Updates and news
  • Important information

Notifications can be local (scheduled on device) or remote (via Firebase). Users can disable notifications anytime from device settings.

4. Quiz Functionality

The app includes a quiz that helps users discover suitable educational profiles.

  • Answers are processed anonymously
  • No personal identity is linked to results
  • No personal profiling is stored

5. Admin Access

The app includes an admin panel accessible only to authorized users.

  • Used to send and schedule notifications
  • Secured via Firebase Authentication

6. How We Use Information

We do not sell or share user data for marketing.
